# TDDD27_2025_simhe960_guskr150_teohe874

Framwork: NextJS

Vi vill göra ett kortspel på webben, där vi hämtar information i en databas med dessa kort. 
Spelmekaniken är baserat på pokemon/hearthstone/magic men simplare. Användaren ska kunna söka i 
databasen och sortera kort baserat på namn, fadderier och krafter. Varje kort kommer även ha ett 
lösenord som gör att man kan spela med det. Vi har en ambition om att kunna göra konton till varje
användare där de kan spara sina kort. Spelare ska kunna söka eller hitta varandra online för att köra
matcher. Spelet kommer vara turnbased.

Vi är föreningsaktiva på campus Norrköping i två olika fadderier där vi ska göra respektiva 
hemsidor till våra fadderier till nästa mottagning. Alla fadderier har fysiska spelkort som de ger ut till
nya studenter under mottagningsperioden. Våran ide är att få det spelet till webben.


Att göra:
1. Matchmaking
2. Spelmekanik
3. Filtrera kort
4. CSS / Grafisk profil
